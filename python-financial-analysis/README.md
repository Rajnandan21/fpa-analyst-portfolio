# Python Financial Analysis

An automated financial ratio and cash flow analysis tool built in Python — calculates liquidity, profitability, leverage, and efficiency ratios from raw financial statement data, then auto-generates charts and a summary report.

## Overview
This project takes 5 years of financial statement data (FY2022A–FY2026E) for an illustrative UK FMCG distribution business and produces a complete automated analysis: ratio calculations, cash flow quality assessment, 4 charts, and a Markdown report — all generated by running one script.

## What It Calculates
**Profitability:** Gross Margin, Net Margin, ROA, ROE
**Liquidity:** Current Ratio, Quick Ratio
**Leverage:** Debt-to-Equity, Interest Coverage
**Efficiency:** Asset Turnover, Inventory/Receivables Turnover, DSO/DIO/DPO, Cash Conversion Cycle
**Cash Flow:** Operating Cash Flow, Free Cash Flow, FCF Margin, FCF-to-Net-Income conversion

## Key Insight (auto-generated)
The current ratio crossed above 1.0x in FY2024A — marking a shift from a working-capital-constrained to a working-capital-healthy position, consistent with the improving cash conversion cycle and falling leverage over the same period. The business converts ~83% of net income into free cash flow on average, indicating high earnings quality.

## Project Structure
| File | Purpose |
|---|---|
| `generate_data.py` | Creates the sample 5-year financial statement dataset |
| `financial_ratios.py` | Calculates all liquidity/profitability/leverage/efficiency ratios |
| `cash_flow_analysis.py` | Calculates free cash flow and conversion metrics |
| `generate_report.py` | Orchestrates everything: builds 4 charts + writes the final Markdown report |
| `financial_statements.csv` | Input data |
| `financial_analysis_report.md` | **The final output** — full report with embedded charts |

## Tools Used
- Python (pandas for calculations, matplotlib for charting)
- Fully modular — each script can be imported and reused independently

## How to Run
```bash
python3 generate_data.py       # creates financial_statements.csv
python3 generate_report.py     # runs ratios + cash flow + charts + report
```
Output: 4 PNG charts and `financial_analysis_report.md` in the same folder.
