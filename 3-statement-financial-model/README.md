# 3-Statement Financial Model

An integrated financial model linking the Income Statement, Balance Sheet, and Cash Flow Statement, built to project company performance and support decision-making.

## Overview
This model demonstrates how the three core financial statements connect — changes in revenue and costs flow through to net income, retained earnings, and ultimately cash position.

## Tools Used
- Excel (formulas, scenario analysis)
- Financial modeling best practices (driver-based assumptions)

## Key Features
- Revenue and cost driver assumptions
- Automated linkages between statements
- Scenario/sensitivity analysis
